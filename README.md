# CEL-Experiment-10-23bme018
Euler's Method is one of the simplest numerical techniques used to solve Ordinary Differential Equations (ODEs). This MATLAB code implements Euler's Method to solve the ODE y' = 4e^(0.8t) - 0.5y with an initial condition of y(0) = 2 over the interval t = 0 to 4 with a step size of h = 1. The method works by estimating the slope at the current point and stepping forward using the formula y(i+1) = y(i) + f(t,y) × h, repeating this process until the end of the interval is reached. At each step, the computed value is printed in a table and stored for plotting, giving a visual representation of how the solution progresses over time. Although Euler's Method is easy to understand and implement, it has a limitation — since it uses the slope only at the beginning of each interval, the error accumulates with every step, reaching approximately 24% by t=4. For problems requiring higher accuracy, more advanced methods such as the Modified Euler Method or the Runge-Kutta Method are recommended.
